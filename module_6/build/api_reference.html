Route Summary

  • GET /: dashboard alias.

  • GET /analysis: renders analysis page.

  • POST /pull-data: starts pull job (foreground or background).

  • POST /update-analysis: refresh trigger endpoint.

  • GET /pull-status: pull status snapshot for polling.
